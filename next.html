Domains

Definition

Domain is a unique name used to identify a website on the internet. It refers to the URL people type in a web browser’s address bar to access your site. Technically, a domain is a textual address that represents the IP address of a server connected to the internetthat provides a user-friendly way for people to navigate the internet without having to remember complex numerical IP addresses. Without domain names, users would need to use a numerical label (IP address) assigned to every website and server on the internet.


Domain Extensions

Domain extensions or also called as top-level domains (TLDs) are the letters that follow the final period in a URL. As the name suggests, TDLs claim the highest spot on the internet’s DNS (domain name system) hierarchy. There are hundreds of TLDs available on the web, which are maintained and regularly updated by the IANA (Internet Assigned Numbers Authority).

There are 4 types:

1. gTLDs (Generic Top-Level Domains)

The most commonly used domain extensions as they fit various website purposes. This includes:
• .com – The most popular extension, originally for commercial websites
• .org – Traditionally used by organizations and non-profits
• .net – Initially intended for network-related sites
• .xyz, .biz, and .tech – Newer alternatives offering more flexibility

2. sTLDs (Sponsored Top-Level Domains)

These domains are restricted to certain organizations and groups. Common examples include:
• .gov for governmental sites
• .edu for educational institutions
• .post for postal services

3. ccTLDs (Country Code Top-Level Domains)

These indicate a specific territory or location using ISO country codes. For example:
• .us for the USA
• .in for India
• .es for Spain
• .id for Indonesia
• co.uk for the United Kingdom

4. Infrastructure Top-Level Domain

This consists of only one domain name:
• .arpa – Representing the Address and Routing Parameter Area, primarily reserved for solving internet-infrastructure issues
